# ☕ Friends Trivia — Central Perk Edition

Un quiz a tema **Friends** con 735 domande su 5 livelli di difficoltà, reaction dal telefilm, musica originale e citazione dell'episodio per ogni risposta.

🎮 **[Gioca ora](https://andrearicciotti1.github.io/FRIENDS-TRIVIA/)**

## Caratteristiche

- **735 domande** su 5 livelli: Easy ☕ · Medium 🛋️ · Hard 🦎 · Expert 📺 · MANIAC 💡
- **Citazione episodio + titolo** su ogni risposta (giusta o sbagliata)
- **Reaction dal telefilm** dopo ogni risposta e a fine livello
- **Musica**: la sigla parte all'ingresso, stacchetti originali tra le domande
- Controlli audio: play / pausa / muto / volume
- Timer per domanda, streak counter, review degli errori
- Funziona su **tutti i sistemi operativi** (è un sito web) e anche **offline**

## Come metterlo online su GitHub Pages

1. Carica tutti i file di questa cartella nel repository (mantieni le sottocartelle `img/` e `audio/`)
2. Vai su **Settings → Pages**
3. In "Source" scegli il branch `main` e cartella `/ (root)`, salva
4. Dopo ~1 minuto il sito è online su `https://andrearicciotti1.github.io/FRIENDS-TRIVIA/`

## Struttura

```
index.html              → pagina principale
style.css               → stile (mood anni '90 / Central Perk)
app.js                  → logica del gioco
data.js                 → 735 domande + titoli episodi
img/reactions/          → reaction dal telefilm
img/bg/                 → sfondi
audio/theme.mp3         → sigla (silenzio iniziale già tagliato)
audio/transitions/      → 8 stacchetti tra le domande
```
